Introduction¶

If you are an app developer you know that keeping your users engaged with your app is very important.

QuantumGraph provides a user engagment platform for mobile using which you can send relevant, timely and personalized push notifications to your users.

This technical documentation details how to integrate QuantumGraph SDK with your app and how to use our web UI to send notifications.
